# Windows 安装包使用说明

## 安装步骤

1. **下载安装包**
   - 下载 `ResumeGenerator-Setup-v1.0.0.exe`

2. **运行安装程序**
   - 双击 `.exe` 文件
   - 按向导提示完成安装
   - 安装程序会自动检测 Node.js 环境

3. **首次运行**
   - 安装完成后会自动检查环境并安装依赖
   - 如果未安装 Node.js，会提示下载

## 使用方法

### 方式一：开始菜单
- 点击「开始」→「简历生成器」→「简历生成器」

### 方式二：桌面快捷方式
- 双击桌面「简历生成器」图标

### 方式三：命令行
```bash
resume-gen create    # 创建新简历
resume-gen build     # 使用已有数据生成
resume-gen templates # 查看模板
```

## 卸载
- 控制面板 → 程序和功能 → 卸载「简历生成器」

## 注意事项

1. **需要 Node.js**：首次使用需要安装 Node.js（安装程序会引导）
2. **依赖自动安装**：首次运行会自动安装 npm 依赖
3. **输出位置**：生成的简历保存在安装目录的 `output` 文件夹

## 系统要求

- Windows 10/11
- Node.js 16+（会自动检测）
- 约 200MB 磁盘空间
